# Acala 网络

Acala是Polkadot中继链的一个平行链，其网络安全型以及共识机制由Polkadot验证人集合提供。作为基于 [Substrate](https://www.substrate.io)的链， Acala大部分的集成要素与Kusama和Polkadot是一致的。作为一个平行链，还是存在很多关键因素上的不同，本文档将进行阐释。
